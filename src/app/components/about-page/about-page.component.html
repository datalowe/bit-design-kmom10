This is Bew Gorp

Ever since I started fiddling with HTML on my sister's Mac in junior high, I knew that the web was for me. I still also fondly remember when I, later on, first installed Xubuntu and figured out that my go-to dev environment would always be Linux-based.

I finished my master's degree in Computer Science at Gothenburg University in 2018. I focused on web programming and operating systems in particular. Since then, I've worked on varying projects for different companies, and contributed to a few open source initiatives as well!

My main interests still lie in everything web related, whether it be coding up an Angular frontend application, putting together a backend with Django, or orchestrating microservices with Docker and Kubernetes. I find that the best part of this job is how many skilled people you get to talk to, and that there's always more to learn.
